Contributions are welcome! 

To create a blog-style post as a "weeknote", add it to the `_posts` directory following this naming convention:

* [YYYY-MM-DD]-[TITLE].md

Look at one of the existing posts for the metadata it should have. Once you're happy with the content, you're free to push directly to the `main` branch, or via pull request if you'd like feedback from colleagues.
