# Welcome

This is our RSE group website repository. It's a clone of [Academic Pages](https://academicpages.github.io/) - a minimal Hugo website intended for communicating about research.

This repository is best viewed [as a website](https://ukceh-rse.github.io/).

## Weeknotes

The main reason this exists is to publish weekly summaries or "weeknotes" talking about what we've been up to.

These might cover group or project activity, or could even be entirely personal!

We don't have a review cycle set up for these. To add a weeknote, create a new markdown file in the `_posts` directory. It should be named in the format `YYYY-MM-DD-page-title.md`. Look at examples in that directory to see what metadata it's helpful for the pages to have.
